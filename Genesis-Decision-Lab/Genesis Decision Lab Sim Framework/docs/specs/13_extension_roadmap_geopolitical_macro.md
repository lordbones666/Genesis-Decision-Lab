# Extension Roadmap: Geopolitical + Macro Enhancements

This module set adds practical bridges beyond baseline binary event updates:

1. **Range forecasting for numeric outcomes**
   - Convert numeric targets into threshold/event probabilities.
   - Provide histogram-style probability distributions from simulated paths.

2. **OSINT tactical signal ingestion**
   - Ingest fast signals (ADS-B/AIS/NOTAM-style) as low-weight evidence.
   - Emit normalized notes and weights for ledger visibility.

3. **Long-range structural baselines**
   - Maintain slow-moving structural offsets and blend with event layer.

4. **Question compiler for vague prompts**
   - Convert vague prompts into resolvable binary question sets.

5. **Event-network propagation**
   - Propagate shocks through causal links to model escalation chains.

6. **Surprise + misinformation filters**
   - Flag surprise indicators and downweight low-credibility signals.

7. **Market/macro bridging helpers**
   - Regime detection + positioning risk for macro-aware adaptation.

These extensions are intentionally modular and can be enabled per-domain via policy registry/config gating.
