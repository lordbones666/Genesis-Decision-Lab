# Abacus Forecasting Spec (V1)

Defines the binary prequential forecasting frame, log-odds representation, append-only evidence stream, deterministic/versioned SEO extraction, optional regime conditioning, proper scoring targets, and minimal replayable schema requirements.

Operationalized in this repo via:
- append-only ledgers
- question gating + validation
- log-odds updates + optional regime layer
- scoring/calibration modules
